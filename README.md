# Сборщик команд (Genshin Team Builder)

GUI-приложение для автоматического составления отрядов в игре Genshin Impact на основе пула персонажей пользователя. Написано на Python с использованием библиотеки CustomTkinter.

## 🚀 Основные возможности

*   **Удобный интерфейс:** Выбор имеющихся персонажей через визуальные карточки.
*   **Умная генерация:** 4 режима подбора отрядов:
    *   *Случайный* — случайная сборка по шаблону.
    *   *Лучший из лучших* — приоритет персонажам с наивысшим рангом.
    *   *Профи* — продвинутый алгоритм, учитывающий элементальные синергии, фракции (например, Нод-Край) и специфичные требования дамаггеров (Навия, Итто, Варка и др.).
    *   *Моноэлемент* — сборка команды из одной стихии.
*   **Система профилей:** Локальное сохранение инвентаря пользователя и истории сгенерированных пачек в JSON-файлы.
*   **Статистика:** Просмотр аналитики аккаунта (распределение по стихиям и редкости).

## 🛠 Технологии

*   **Язык:** Python 3.10+
*   **Интерфейс:** CustomTkinter, Pillow (для обработки изображений)
*   **Данные:** JSON
*   **Сборка:** PyInstaller

## ⚙️ Установка и запуск (для пользователей)
1. Зайдите в раздел "Releases";
2. Выберите самую новую версию;
3. Скачайте файл с программой (Genshin Command Maker.rar);
4. Распакуйте архив в любом удобном для вас месте;
5. Запускайте файл Genshin Command Maker.exe.

## ⚙️ Установка и запуск (для разработчиков)

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/VolentCry/genchin_command_maker.git](https://github.com/VolentCry/genchin_command_maker.git)
   ```
2. Установите зависимости:
  ```bash
  pip install customtkinter Pillow
  ```
3. Запустите проект:
  ```bash
  python main_window.py
  ```
